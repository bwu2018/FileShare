dnsfileshare

Upload a file to share it over DNS. Your browser encrypts it and does all the chunking and hashing locally -- nothing is sent to any server of ours in plaintext, and no server of ours ever sees the decryption key. You'll get a link back; anyone with that link can view and download the file. There's no way to delete a published file once it's shared, so only upload what you're comfortable leaving public indefinitely.
